# @ROOMページ表示画像 貼り付け用HTML

授業ページに、@ROOMページ表示回数画像を貼り付けるための `<img>` タグ一覧です。

この画像は、StudyRoomの入室人数ではなく、該当する授業回ページが表示された回数を示します。第1講から第4講までが同じ授業回に属する場合は、同じ授業回の画像を貼り付けます。

## 説明文案

この画像は、この授業回のページが最近どれくらい表示されたかを示しています。
StudyRoomの入室人数とは別に、同じ授業回を開いている人の気配を確認できます。

授業を始める前に、よければStudyRoomにもチェックインしてみてください。
リンクをクリックするだけで、60分間この授業回を学習中の学生として表示されます。
アカウントの作成やログインは不要です。

「科目名 第〇回」StudyRoomにチェックインする（新しいタブが開きます）
